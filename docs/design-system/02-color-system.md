# Color System

## Purpose

The color system establishes a consistent visual language across the portfolio. Colors communicate hierarchy, interaction, feedback, and brand identity.

---

## Color Categories

### Brand

Core identity colors.

- Primary
- Secondary
- Accent

---

### Neutral

Used for text, backgrounds, borders, and surfaces.

- 50–950 scale

---

### Semantic

Communicate interface states.

- Success
- Warning
- Error
- Info

---

### Surface

Application backgrounds.

- Background
- Surface
- Elevated Surface

---

### Content

Text hierarchy.

- Primary Text
- Secondary Text
- Muted Text
- Inverse Text

---

### Border

Visual separation.

- Default
- Strong
- Focus

---

## Principles

- Colors communicate meaning.
- Contrast always comes first.
- Use accent colors sparingly.
- Neutrals should dominate the interface.
- Every color must have a defined purpose.

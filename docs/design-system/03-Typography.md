# Typography

## Purpose

Typography establishes hierarchy, readability, and personality across the portfolio. It should feel modern, elegant, and effortless to read.

---

## Principles

- Readability comes first.
- Limit the number of font families.
- Use typography to create visual hierarchy.
- Maintain consistent spacing between headings and content.
- Typography should reinforce the Editorial Engineering identity.

---

## Font Roles

### Display

Large titles and hero sections.

### Heading

Section titles and component headings.

### Body

Paragraphs and long-form content.

### Label

Buttons, navigation, tags, and UI elements.

### Code

Code snippets and technical content.

---

## Font Families

Typography uses three complementary font families to reinforce the Editorial Engineering identity.

### Display

Used for hero sections and large brand statements.

Font:

- Playfair Display

Purpose:

- Create personality.
- Add editorial character.
- Establish visual identity.

---

### Body

Used for paragraphs, descriptions, and general UI content.

Font:

- Inter

Purpose:

- Maximize readability.
- Maintain a modern product feel.
- Support long-form content.

---

### Code

Used for technical content, tags, and developer-focused details.

Font:

- JetBrains Mono

Purpose:

- Reinforce engineering identity.
- Create technical contrast.

## Tokens

Typography tokens should expose:

- Font families
- Font weights
- Font sizes
- Line heights
- Letter spacing

Components should consume semantic typography tokens instead of defining typography values directly.

---

## Type Values

Typography tokens define the following values:

| Token      | Size | Line Height |
| ---------- | ---- | ----------- |
| Display XL | 72px | 1.05        |
| Display LG | 56px | 1.1         |
| Heading LG | 40px | 1.2         |
| Heading MD | 32px | 1.25        |
| Heading SM | 24px | 1.3         |
| Body LG    | 20px | 1.6         |
| Body MD    | 16px | 1.6         |
| Body SM    | 14px | 1.5         |
| Label MD   | 14px | 1.4         |
| Label SM   | 12px | 1.4         |

---

## Accessibility

- Ensure sufficient contrast.
- Avoid excessively small text.
- Maintain comfortable line height.
- Support responsive scaling.

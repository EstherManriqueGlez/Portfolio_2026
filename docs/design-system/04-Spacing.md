# Spacing

## Purpose

Spacing creates rhythm, balance, and consistency across the interface. It improves readability and helps users understand relationships between elements.

---

## Principles

- Use a consistent spacing scale.
- Favor whitespace over visual clutter.
- Maintain predictable vertical rhythm.
- Components should share the same spacing rules.
- Layout spacing should scale across screen sizes.

---

## Spacing Scale

The design system uses a spacing scale based on consistent increments.

Examples include:

- XS
- SM
- MD
- LG
- XL
- 2XL
- 3XL

---

## Usage

### Component Spacing

Internal padding and gaps.

### Layout Spacing

Distance between sections and major blocks.

### Content Spacing

Spacing between headings, paragraphs, lists, and media.

---

## Responsive Behavior

Spacing increases progressively on larger screens while preserving visual rhythm.
